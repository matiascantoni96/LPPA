Variables, tipos de datos y operadores

Variables y tipos de datos

  • Las variables se definen con la palabra clave var
  • Las variables son case sensitive por lo cual var A no es igual que var a
  • Los tipos de datos posibles en javascript son number, string, boolean, object y unedfined

Operadores

  • De asignación: =
  • De comparación: == === != !== > < >= <=
  • Aritméticos: + - * / %
  • Lógicos: && || !
  • De cadena de caracteres: +
  • La lista completa MDN
